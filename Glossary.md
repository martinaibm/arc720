# Glossary

## Salesforce Terms

- **Apex**: A strongly typed, object-oriented programming language that allows developers to execute flow and transaction control statements on the Salesforce platform.
- **Lightning**: A component-based framework for developing dynamic web apps for mobile and desktop devices on the Salesforce platform.
- **Trailhead**: An online learning platform that offers courses for Salesforce and other related technologies.
  
## MuleSoft Terms

- **API**: A set of rules that allow different software applications to communicate with each other.
- **Anypoint Platform**: A unified integration platform that provides a complete set of tools for API management, design, and integration.
- **RAML**: RESTful API Modeling Language, a language used to describe RESTful APIs.
