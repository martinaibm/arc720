# Anypoint MQ

## What is / Description
Is a message queuing service for exchanging data asynchronously between our applications, based in cloud. Enables applications to communicate by publishing a Mule message to a queue so that another app can consume the message. Supports large payloads, encryption, queuing, and publish/subscribe (pub/sub) paradigms. Enables to easily connect to non-Mule applications by using the REST API.
